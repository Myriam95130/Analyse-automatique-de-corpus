# Étude lexicographique comparative de corpus centrée sur les verbes

# Corpus :
1. Verne (Jules) Le tour du monde en quatre-vingt jours
2. Verne (Jules) Les Révoltés de la "Bounty"
3. Stendhal Mémoires d'un touriste (Voyage en Bretagne et en Normandie) (1838)

# Objectif :
Réalisation d'une étude lexicographique comparative de corpus, centrée sur les verbes en se
basant sur les textes du corpus et le lexique dimaju-4.1.1-utf8.txt (étiquettes). La plupart
des étapes vont nécessiter l'écriture de programmes Python mettant en jeu les instructions/structures de données Python.

*Projet réalisé dans le cadre du cours de Traitement Automatique de Corpus de L3 TAL à l'INALCO (Enseignant : P.Paroubek).*
